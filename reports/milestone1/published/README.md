# Published evidence contract

Published evidence is small, sanitized, and source-linked. It contains summaries
only; it does not contain weights, model caches, credentials, hostnames, private
paths, complete logs, or unverified claims.

The expected files are `environment_summary.json`, `runtime_summary.json`,
`baseline_summary.json`, `trace_summary.json`, `source_map.json`, and
`FINDINGS.md`. Each file is generated only after the target-machine validation
has succeeded.
